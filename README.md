# 🌳 Syntax Tree Generator (NLP Web App)

A Streamlit-based Natural Language Processing (NLP) web application that generates and visualizes syntactic dependency trees from English sentences using spaCy and Graphviz.

---

## 🚀 Live Demo
Try the app here:  
👉 https://huggingface.co/spaces/Ravi-Manghwar/Syntax-Tree-Generator

---

## 📸 Project Overview
This project takes an English sentence as input and generates a visual syntax (dependency) tree. It also shows grammatical relationships between words such as subject, object, modifiers, etc.

---

## ✨ Features
- 🧠 Dependency parsing using spaCy NLP model
- 🌳 Interactive syntax tree visualization
- 📊 Graph-based representation using Graphviz
- 💬 Word-level grammatical relations
- ⚡ Automatic model loading (en_core_web_sm)

---

## 🛠️ Technologies Used
- Python
- Streamlit
- spaCy
- Graphviz
